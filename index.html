Kort fortalt

En dyst med poster, point og fælles mission.

Heltedysten kan bruges som ramme for et løb, en temadag eller en digital aktivitet. Holdene får opgaver, løser udfordringer og samler point frem mod en samlet afslutning.

Siden er 100% statisk og kan derfor hostes direkte på GitHub Pages uden database, login eller server.

For arrangører

Opdater dato, kontaktinformation og FAQ før siden lægges live på GitHub Pages.

FAQ

Ofte stillede spørgsmål

Hvad er Heltedysten?

Heltedysten er en eventramme med opgaver, poster og point. Indholdet kan tilpasses jeres egen lejr, gruppe eller aktivitet.

Skal deltagerne bruge telefon?

Kun hvis I vælger det. Siden kan bruges som informationsside, men e-ventet kan også køres med QR-koder, formularer eller helt analogt.

Hvordan ændrer jeg datoen?

Åbn index.html, find data-event-date og skift værdien til jeres dato og tidspunkt.

Kan siden hostes på GitHub Pages?

Ja. Upload filerne til et repository, aktiver GitHub Pages og behold CNAME-filen, hvis domænet skal være heltedysten.dk.
